# Security-System-Part-1
Designed a security system that is basic in which an alarm starts ringing as soon as the door is opened, this is the first to developing a much better system in which I plan on using more advanced materials. For part 1 I used Arduino Uno R3, Breadboard, Magnetic Reed Switch, Magnet, Active Buzzer, Resistor, Jumper Wires, USB Cable, Foam Board/Styrofoam House Model
